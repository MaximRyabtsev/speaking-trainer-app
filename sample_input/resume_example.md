# Example resume text

I am a backend developer with experience building Python services, data pipelines, and internal automation tools. I have worked on projects involving API integrations, PostgreSQL databases, background workers, and monitoring dashboards.

The most difficult part was handling unreliable external APIs and making the pipeline robust enough to recover from partial failures.

I also helped improve deployment processes by creating scripts that reduced manual release steps. This made releases faster and less error-prone.

I want to improve my ability to explain my work clearly in English, especially when talking about technical decisions, challenges, trade-offs, and measurable impact.
